常用配置属性

  • title 标题组件
  • tooltip 提示框组件
  • legend 图例组件
  • toolbox 工具栏
  • grid 直角坐标系内绘图网格
  • xAxis 直角坐标系 grid 中的 x 轴
  • xAxis 直交坐标系 grid 中的 y 轴
  • series 系列列表
  • color 调色颜色列表

临时扩展css

边框图片

属性 描述
border-image-source URL
border-image-slice 图片边框内响内偏移
border-image-width 图片边框的宽度
border-image-repeat 图像边框是否应 repeat（平铺）、round（铺满）、stretch（默认拉伸）
