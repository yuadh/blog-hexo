兄弟组件通信

  • 将共享状态提升到最近的公共父组件中，由公共父组件管理这个状态
  • 思想：状态提升
  • 公共父组件职责：提供共享状态、提供操作共享状态的方法
  • 要通信的子组件只需通过 props 接收状态或操作状态的方法

context跨组件

实现思路

  • 调用 React.createContext() 创建 Provider 和 Consumer 两个组件

  • 使用 Provider 组件组为父节点

  • 设置 value 属性，表示要传递的数据

  • 调用 Consumer 组件接收数据

    注意：Provider和Consumer 的使用是一对的
